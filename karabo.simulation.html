karabo.simulation.observation module

class Observation(start_frequency_hz: float, start_date_and_time: datetime = datetime.datetime(2022, 6, 24, 9, 59, 11, 731419), length: timedelta = datetime.timedelta(seconds=43200), number_of_channels: float = 1, frequency_increment_hz: float = 0, phase_centre_ra_deg: float = 0, phase_centre_dec_deg: float = 0, number_of_time_steps: float = 1)

Bases: object

The Observation class acts as an object to hold all important information about an Observation.

Required parameters for running a simulation with these observational settings.

Variables

  • start_frequency_hz – The frequency at the midpoint of the first channel, in Hz.

  • start_date_and_time – The start time and date for the observation. Default is datetime.utcnow().

  • length – Length of observation. Default is 12 hours.

Optional parameters for running a simulation with these observational settings.

Variables

  • number_of_channels – Number of channels / bands to use

  • frequency_increment_hz – The frequency increment between successive channels, in Hz.

  • phase_centre_ra_deg – Right Ascension of the observation pointing (phase centre), in degrees.

  • phase_centre_dec_deg – Declination of the observation pointing (phase centre), in degrees.

  • number_of_time_steps – Number of time steps in the output data during the observation length. This corresponds to the number of correlator dumps for interferometer simulations, and the number of beam pattern snapshots for beam pattern simulations.
